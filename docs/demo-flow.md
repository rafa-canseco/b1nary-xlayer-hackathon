# Demo Flow

## Setup

1. Start backend with XLayer config.
2. Start market maker with a funded and whitelisted MM wallet.
3. Start frontend with `.env.xlayer.example` copied to `.env.local`.
4. Open the frontend locally.

## Recording Script

1. Show the app connected to XLayer testnet.
2. Click the faucet action to receive test OKB and USDC.
3. Open the OKB options market.
4. Show executable PUT and CALL quotes.
5. Accept one quote.
6. Open the transaction on the XLayer testnet explorer.
